# Olympus Cyber

Olympus Cyber is a draft app pack built from the Dennco Olympus Command master platform.

## Purpose

Cybersecurity alerting, vulnerability awareness, vendor advisories, asset risk context, incident notes, and executive cyber reporting.

## Likely users

- IT administrators
- Security analysts
- Managed service providers
- Executives
- Compliance reviewers

## Starting modules

- Cyber alert feed
- NVD / vulnerability source lane
- CISA KEV source lane
- Vendor advisory tracking
- Asset and vendor risk views
- AI-assisted cyber briefs
- Executive report builder

## App boundary

This app should support lawful defense, monitoring, source-cited reporting, and organization-approved cyber operations. It should not include offensive exploitation, credential theft, malware, unauthorized access, or evasion tooling.

## Status

Draft planning scaffold. Not production-ready.
